# 도구 아이디어 초안

## 한 줄 요약
엑셀로 관리하던 **대회 종목별 참가자 정보**를 회사 전체가
실시간으로 보고 수정하며 이력까지 관리하는 도구

## 대상 사용자
- 회사 전체 (부서 구분 없이 모든 직원)

## 현재 상황 (AS-IS)
- 엑셀 / 구글 시트로 수동 관리 중
- 사람이 직접 입력하다 보니 실수·오류 빈번

## 기대 효과 (TO-BE)
- 실수 최소화: 입력 자동화 및 유효성 검사로 오류 차단
- 시간 절약: 반복 입력 작업 제거
- 실시간 공유: 팀 전체가 동일한 최신 데이터를 동시에 확인
- 이력 추적: 누가 언제 무엇을 바꿨는지 기록

## 핵심 기능 (초안)
- 대회 종목별 참가자 등록 / 수정 / 삭제
- 실시간 다중 사용자 접근 (회사 전체)
- 변경 이력 자동 저장
- 입력 오류 방지 (유효성 검사)
